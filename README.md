# sharele

sharele 是一个“移动职业/兴趣角色地图”Web 平台。

## 核心流程
1. 用户进入网站
2. 登录
3. 实名认证
4. 选择角色（如移动摄影师、移动化妆师、移动模特、移动小吃摊、移动骑友、移动登山客等）
5. 进入地图查看附近/在线的对应角色人群

## 当前阶段
- 需求确认 + 技术方案设计中
- 目标端：Web 优先，后续嵌入手机 App（需移动端适配）
